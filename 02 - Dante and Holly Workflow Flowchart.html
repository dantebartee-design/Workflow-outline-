File naming is set up front

Dante names the file the moment it's created — date, description, and his initial (D). Every later touch appends an initial, building the revision trail.

Two approval gates

Dante's "ready for Holly?" check, and Holly's "approve?" decision. Nothing moves forward until the person owning that gate signs off.

Folder moves mark progress

To Be Disclosed → Disclosures (+ Attachments) → Done. The folder a document sits in always tells you its stage.

Clio is the final destination

Only the approved, finalised document is exported to Clio for filing/serving — the clean system of record.
