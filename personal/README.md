# Personal

Personal workspace. Four projects:

- **[Disclosure Strategy](./disclosure-strategy/)** *(Chat Project)* — vision and
  approach for disclosures, plus the post-case **debriefs**. The "why" and
  "what", before any case is run; the "what I learned", after.
- **[Personal Explorations](./personal-explorations/)** *(Chat Project)* —
  open-ended thinking unrelated to a specific disclosure run.
- **[Workflows & Architecture](./workflows-and-architecture/)** *(Cowork Project)*
  — the reusable, polished playbook library that every Teams case draws from.
- **[Build & Debug](./build-and-debug/)** *(Cowork Project)* — the messy sandbox
  where disclosure code is written and debugged, then **promoted** into the
  Workflows & Architecture library.
