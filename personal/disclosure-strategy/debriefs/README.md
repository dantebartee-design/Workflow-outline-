# Debriefs

The "feelings & learning" conversations that come **home to Personal** after a
Teams case closes. The audit stays in Teams (*did it work?*); the debrief lives
here (*what did I learn, how did it feel?*).

This is the hinge of the learning loop:

```
Teams: Audit  →  Personal: Debrief  →  improves the Workflows & Architecture library  →  next case starts better
```

## How to use

1. When a case's audit closes, copy the debrief template
   [`../../workflows-and-architecture/playbooks/debrief.md`](../../workflows-and-architecture/playbooks/debrief.md)
   into this folder as `debrief-<case-name>.md`.
2. Fill it in honestly — including how it felt, not just what happened.
3. For every lesson, **edit the relevant playbook** in
   `personal/workflows-and-architecture/playbooks/`. That edit is what closes
   the loop.
